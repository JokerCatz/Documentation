# Scaffold

Scaffold is a PHP web framework that specializes in RESTful APIs. 

## Requirements

 * PHP >= 5.4
 * For database access, a SQL-based server supported by [PDO](http://php.net/manual/en/pdo.drivers.php).
 * Apache server with mod_rewrite enabled.
